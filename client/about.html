About UrbanEye

Event Check & Report

Main Idea

UrbanEye is a web-based platform that empowers users to report and track events near their location, helping communities stay informed about local events (including crime, traffic, street entertainment). The platform allows users to share real-time reports, browse reported events, and engage in discussions.

Core Features

Landing Page

Explains the purpose and functionality of the website.

Event Browsing

Users without an account can check events but have limited functionality.

Event Listing

List all events reported with title, location and time. Filters/Search allow for the list of events to be specified (filter by time, type).

Event Details

Show title, timeline, photo. Users can make comments and like events.

Report Event

Users can report events by uploading title, location, and photo.

User Profile

Created when a user registers. Displays their username, profile picture. Retains a list of all events reported by the user.

Planned Enhancements

Interactive Map View

Show all events on a map, display distance from user using Google Maps/Mapbox JS API.

External Database Integration

Access to open crime/offender's databases for additional information.

Trends Analytics

A chart showing the number of events per day to identify patterns.

Event Verification

Allow users to report events that they think are not true.

Personalized Recommendations

Show events that users are most interested in on the first page (analyzed by past click record).

Nearby Events

Show other popular (by event click time) events near the current event on the event details page.

AI Assistant

Summarize events, answer questions about specific events, analyze real-time event data to summarize past trends and predict future event numbers.
